# AppOS
Web技术实现的应用层面操作系统。

## Feature
- 最新Web技术试验田，永不兼容IE，也不会做跨浏览器兼容
- 基于Polymer的Web Component开发模式，使用bower进行包依赖管理
- 基于Pure.css布局样式
- 图标采用iconfont.cn平台支持
- 支持自定义插件扩展
